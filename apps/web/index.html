History Rules

How CWA matching works

Registry match key

CWA rows are matched to the registry with the student index number, then carried into academic history and latest registry CWA.

Semester-aware timeline

Each import keeps the academic year and semester label so later searches can show the most recent CWA without losing older records.

Name mismatch warnings

If the workbook name differs from the registry name, the row stays valid but is surfaced for reviewer attention before import.

Workbook flexibility

The importer reads context from the workbook and sheet text, so repeated result blocks and college-grouped files can still be merged safely.
